Docker版本升级详细步骤

一、前言

docker的版本分为社区版docker-ce和企业版dokcer-ee社,区版是免费提供给个人开发者和小型团体使用的，企业版会提供额外的收费服务，如果你的机器上安装了老版本的docker，那么就需要卸载，本文介绍如何完全的卸载老版本以及安装新版本docker。

二、删除老版本

2.1 查看当前版本

rpm -qa | grep docker

docker升级之前已存在的image

2.2 停止docker服务

systemctl stop docker

停止docker服务报错如下图：

解决：

systemctl daemon-reexec

2.3 卸载docker软件包

sudo yum remove docker \
              docker-client \
              docker-client-latest \
              docker-common \
              docker-latest \
              docker-latest-logrotate \
              docker-logrotate \
              docker-engine

2.4 删除相关配置文件

find /etc/systemd -name '*docker*' -exec rm -f {} \;
rm -rf /var/lib/docker   #删除以前已有的镜像和容器,非必要
rm -rf /var/run/docker

三、安装新版本

3.1 软件包安装并设置yum源

sudo yum install -y yum-utils

sudo yum-config-manager \
--add-repo \
https://download.docker.com/linux/centos/docker-ce.repo

3.2 安装docker

sudo yum install docker-ce docker-ce-cli containerd.io

3.3 启动并开机自启

systemctl start docker
systemctl enable docker

3.4 查看docker版本

docker --version

四、之前的容器无法启动

解决:

删除当前容器，基于当前的镜像，重新创建容器即可。
